# EveryGrad Template MVP
## 1. Purpose
- To build a data-driven academic resource platform that provides students with ready-to-use, high-quality academic templates 
(projects, SIWES reports, proposals, journals, etc.).
- To empower students who cannot afford AI-generated full reports by giving them accessible, structured learning materials.
- To serve as a data collection and user acquisition product for the broader EveryGrad ecosystem, feeding verified academic 
materials into EveryGrad AI for future training.

## 2. Key Features / MVP
- Template Library 
- AI Pre-Verification Engine
- Researcher Monetization System
- Points & Reward System
- Referral System
- Admin Dashboard

## 3. Tech Stack
Python, Django, MySQL, Redis (etc.)

## 5. Risks / Constraints

- Database scaling
